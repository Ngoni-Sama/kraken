# KRAKEN

The SMS-INTERNET Bridge using RaspberryPi

## Note

A Nice documentation will be provide as soon as possible !

## Using

```
$ cd path/project
$ virtualenv -p python3 kraken
$ source kraken/bin/activate
$ pip install -r dev_requirements.txt
$ python run.py
```

To run the tests:

```
$ pytest tests
```
